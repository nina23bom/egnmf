# EGNMF

EGNMF is a GNMF-based clustering algorithm using cluster ensembles. The algorithm stably achieves a high and robust clustering performance. The usage of the algorithm is shown in simulation.ipynb. 

Dependency
----

- numpy
- scipy
- scikit-learn
- networkx <= 2.3
- metis
- munkres
- pyitlib
